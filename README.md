# Smart Home Dashboard

Ky projekt është një aplikacion web i thjeshtë për menaxhimin e pajisjeve të një Smart Home,
i realizuar si detyrë kursi.

## Përshkrimi
Aplikacioni lejon përdoruesin të:
- Hyrë në sistem përmes një faqe login
- Shikojë dhomat e shtëpisë
- Aktivizojë dhe çaktivizojë pajisje si drita dhe ventilatorë
- Shikojë ndryshime vizuale sipas gjendjes ON/OFF

## Teknologjitë e përdorura
- HTML
- CSS
- JavaScript
- Git & GitHub për versionimin e kodit

## Funksionalitete kryesore
- Login i thjeshtë (front-end)
- Ndryshim vizual i pajisjeve ON/OFF
- Ikona SVG për dritat dhe ventilatorët
- Ndërfaqe e thjeshtë dhe responsive
